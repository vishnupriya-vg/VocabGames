# Word Game 1

An early HTML/JavaScript vocabulary game prototype.

## What's Inside

| File | Description |
|------|-------------|
| `index.html` | Main vocabulary quiz game |
| `parcel-post.html` | Parcel post word game variant |
| `mockup.html` | UI mockup/prototype |
| `mockup-games.html` | Games mockup |
| `questions.js` | Main question bank |
| `parcel-questions.js` | Parcel post question bank |
| `Vocab games- WORD LIST.xlsx` | Source word list spreadsheet |
| `notes.md` | Development notes |
| `claude.md` | Claude Code context file |

## How to Run

No installation needed. Just open `index.html` in your browser.

## Tech Stack

- Plain HTML, CSS, JavaScript
- No build step required
