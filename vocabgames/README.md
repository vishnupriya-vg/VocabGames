# Word Games Monorepo

A collection of vocabulary and word games built for learning. Each game is self-contained in its own folder.

---

## Games

| Folder | Game | Tech | Description |
|--------|------|------|-------------|
| [`vocabgames/`](./vocabgames/) | Word Sprout | React + TypeScript | Full vocab learning app with Word Match, Multiple Choice, and Fill in the Blank |
| [`word-game-1/`](./word-game-1/) | Word Game 1 | HTML + JavaScript | Early prototype with vocabulary quiz and parcel post game |
| [`word-game-2/`](./word-game-2/) | Word Game 2 | HTML + JavaScript | Evolved version of Word Game 1 with improved question set |
| [`word-game-3/`](./word-game-3/) | Word Game 3 | HTML + JavaScript | Standalone vocabulary game |
| [`word-search-puzzle/`](./word-search-puzzle/) | Word Search Puzzle | HTML + JavaScript | Interactive word search puzzle game |
| [`transformation-path-game/`](./transformation-path-game/) | Transformation Path Game | React + Vite | Word transformation challenge game |
| [`hear-it-know-it-spell-it/`](./hear-it-know-it-spell-it/) | Hear It · Know It · Spell It | React + Vite | Audio-based vocabulary game — hear a word, identify its meaning, spell it out. Grades 3–8. |

---

## Running Each Game

### vocabgames (React app)
```bash
cd vocabgames
npm install
npm run dev
```

### word-game-1, word-game-2, word-game-3, word-search-puzzle (HTML games)
Just open `index.html` directly in your browser — no installation needed.

### transformation-path-game (React app)
```bash
cd transformation-path-game
npm install
npm run dev
```

### hear-it-know-it-spell-it (React app)
```bash
cd hear-it-know-it-spell-it
npm install
npm run dev
```
